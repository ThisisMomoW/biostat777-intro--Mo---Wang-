## About this project
This Quarto website was created as part of a data analysis project for the course.  
It explores the famous *Iris* dataset using R, including data wrangling with **dplyr** and visualization with **ggplot2**.

The website includes:
- A home page introducing the author
- An “About” page describing professional interests
- An “Example analysis” page demonstrating exploratory data analysis
